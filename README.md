# Lior Auvgang Landing Page

Static landing page prepared for Vercel.

## Before publishing

1. Open `script.js`.
2. Replace `your-email@example.com` with the real contact email.
3. Optional: replace videos inside `/assets` with higher quality versions if you host them on YouTube/Vimeo.

## Deploy to Vercel

1. Create a free GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel and choose **Add New Project**.
4. Import the GitHub repository.
5. Keep default settings and deploy.

You can also drag this folder into Vercel if your account supports direct upload.
